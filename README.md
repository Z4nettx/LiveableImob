# 🏡 Liveable  
### Sua nova forma de encontrar um lugar para viver

> **Liveable** é uma plataforma web de locação de imóveis pensada para simplificar conexões entre proprietários e inquilinos, com uma experiência intuitiva, moderna e eficiente.

---

## ✨ Visão Geral

O **Liveable** nasce com o objetivo de transformar a forma como as pessoas encontram imóveis para morar. Em vez de processos burocráticos e confusos, oferecemos uma experiência clara, rápida e acessível.

Seja você alguém buscando seu próximo lar ou um proprietário querendo anunciar seu imóvel, o Liveable é o lugar certo.

---

## 🚀 Funcionalidades

- 🔍 **Busca inteligente de imóveis**  
  Encontre imóveis por localização, preço, tipo e outros filtros personalizados.

- 🏠 **Listagem de propriedades**  
  Visualize detalhes completos, imagens e informações relevantes de cada imóvel.

- 👤 **Sistema de usuários**  
  Cadastro e autenticação para proprietários e interessados.

- 📄 **Gerenciamento de anúncios**  
  Proprietários podem criar, editar e remover seus imóveis com facilidade.

- ❤️ **Favoritos**  
  Salve imóveis de interesse para visualizar depois.

- 📱 **Interface responsiva**  
  Acesse de qualquer dispositivo: desktop, tablet ou smartphone.
